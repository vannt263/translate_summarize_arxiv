    "one of the main problems of feynman - diagrammatic computations is the enormous growth of the number of feynman diagrams , not only with the loop order , but also with the number of particles in a model . while many precision calculations in the standard model ( sm ) could still be performed by hand exactly , the same is very difficult in models like the minimal supersymmetric standard model ( mssm ) , even more so when parameters are allowed to take complex values ( cmssm ) .",
    "yet it is highly desirable to perform unabridged calculations in the mssm , too , since also this model allows to make precise predictions in terms of a set of input parameters .    with the availability of powerful software packages ,",
    "the basic problem of bookkeeping and calculation of the diagrams has been solved for many common cases .",
    "still , it is not entirely trivial to code a model of the complexity of the mssm in such a system , since this has to be done in a reasonably general way ( i.e.  not only for special cases of the parameters ) and many checks have to be performed to test all sectors of the model .",
