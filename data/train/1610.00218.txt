    "unmagnetized @xmath32 discharge plasmas solver can be conveniently used to study parametric dependence of local instabilities .",
    "general view of the software interface is shown in fig . [",
    "fig : figa1 ] .",
    "it consists of three main fields : 1  plasma parameters ; 2  define the include effects and displays the solved equation ; 3calculations , plotting and saving functions .",
    "all parameters and their dimensionality are defined in field 1 .",
    "regular or scientific notation can be used for input values .",
    "there are only two types of gases for now : xenon and argon .",
    "electric potential is for the ion acceleration in the @xmath1 direction ; this value is used only if the ion flow effect is included .",
    "there is a check on empty or non - numeric fields ; however , there are no limitations on input numbers .",
    "the solver tool calculates the dependency of growth rate and frequency as a function of the wave numbers @xmath23 or @xmath25 .",
    "checkboxes control along which axis the eigenfrequency will be plotted .",
    "once , one of the checkboxes is checked , the second checkbox becomes disabled ; however , the fixed value of the second wavenumber can be changed .",
    "additional field appears , which is used to indicate the range of the wavenumbers .",
    "[ fig : figa2 ] for detailed view .",
    "calculations will be done for the given range of wavenumbers in one direction and fixed value of the wavenumber in another direction .",
    "field 2 allows the user to change the included effects and solved equations .",
    "the corresponding equation will be shown at the bottom of the field 2 .",
    "field 3 is a control section . to perform calculations ,",
    "the calculation button should be pressed .",
    "plot button will plot the results .",
    "after the results are plotted , they can be saved by save button .",
    "note , that window with plotted results should remain open , so it can be saved .",
    "figure name field is for the name of the output files .",
    "there are two output files : structure with all variables , and eps format figure ."
