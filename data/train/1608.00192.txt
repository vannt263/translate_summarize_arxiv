    "this paper considers the problem of game theoretic control of multi - agent systems .",
    "assume there is a system level objective function , say consensus , then we may design a local information based utility functions such that the multi - agent system becomes a potential game with the system level objective function as the potential function .",
    "then individual agents can use their local information to reach an equilibrium , which maximizes the objective function .",
    "two cases have been investigated .",
    "( i ) fixed topology , and ( ii ) time - varying topology .",
    "necessary and sufficient conditions have been obtained for both cases .",
    "some examples are presented to illustrate the theoretical results .      1 .   the state transition process used in this paper for time - varying topology is designable .",
    "a challenge problem for further investigation is to relax this . 2 .",

    "<S> control of multi - agent systems via game theory is investigated . </S>",
    "<S> assume a system level object is given , the utility functions for individual agents are designed to convert a multi - agent system into a potential game . </S>",
    "<S> first , for fixed topology , a necessary and sufficient condition is given to assure the existence of local information based utility functions . </S>",
    "<S> then using local information the system can converge to a maximum point of the system object , which is a nash equilibrium . </S>",
    "<S> it is also proved that a networked evolutionary potential game is a special case of this multi - agent system . </S>",
    "<S> second , for time - varying topology , the state based potential game is utilized to design the optimal control . </S>",
    "<S> a strategy based markov state transition process is proposed to assure the existence of state based potential function . as an extension of the fixed topology case </S>",
    "<S> , a necessary and sufficient condition for the existence of state depending utility functions using local information is also presented . </S>",
    "<S> it is also proved that using better reply with inertia strategy , the system converges to a maximum strategy of the state based system object , which is called the recurrent state equilibrium .    ,    ,    finite game , ( state based ) potential function , multi - agent systems , designed utility function , semi - tensor product of matrices . </S>"
