    "<S> the first problem is the root problem : decide if the element is a power and in this case compute the roots . </S>",
    "<S> the second problem is the symmetry problem : decide if the element commutes with a finite order element and in this case compute this element . </S>",
    "<S> the structure theorem on which this algorithm is based provides also a new solution to the conjugacy problem . </S>"
